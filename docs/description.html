# 🪄 RGX | Coordination Cloak Utility ![CCU Logo](https://raw.githubusercontent.com/donniedice/CoordinationCloakUtility/main/images/ccu_logo.png) ## ![](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Coordination Cloak Utility** automates the tedious process of equipping, using, and re-equipping teleportation cloaks in World of Warcraft. One click to scan your bags, swap your cloak, teleport, and restore your original gear — all automatically. ![RealmGX Kiwi](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) **The Kiwi Says:** "One click, any destination! Bwwiiiee." *** ## ✨ Features 🔍 Smart Cloak Detection - Automatically finds all teleportation cloaks in your bags! ⚡ One-Click Teleportation - Equip, use, and re-equip with a single button press. 🎯 Auto Gear Swap - Saves and restores your original cloak automatically after teleporting. 🛡️ Combat Safe - Prevents gear changes during combat to keep you compliant. 🚀 Zero Configuration - Works immediately after installation — no setup required! 📌 Smart Button Positioning - Remembers your preferred button location, fully draggable. 💎 Item Priority - Prioritizes highest item level teleportation cloak automatically. ⏱️ Cooldown Aware - Shows remaining cooldown time on the button so you're always informed. 🌍 10+ Languages - Supports all major WoW locales with automatic detection. 💾 Persistent Settings - Settings saved automatically between sessions. **Supported Cloaks:** Cloak of Coordination (Guild), Wrap of Unity (Alliance), Shroud of Cooperation (Horde) — all variants supported! *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/ccu # Toggle the CCU button
/ccu help # Show all available commands
/ccu reset # Reset button position to default
/ccu status # Show current addon status
