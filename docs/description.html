## ![](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | CCU** automates teleport cloak handling in World of Warcraft. Use `/ccu` for the popup button, or click the movable minimap icon to run the cloak flow directly and restore your previous cloak after zoning. ![RealmGX Kiwi](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) **The Kiwi Says:** "One click, any destination! Bwwiiiee." *** ## ✨ Features 🔍 Smart Cloak Detection - Finds supported teleportation cloaks in your bags automatically. ⚡ Reliable Use Button - Uses the equipped back-slot teleport cloak through a secure click action. 🎯 Auto Restore - Re-equips your original cloak after teleporting. 🛡️ Combat Safe - Avoids protected gear swaps while you are in combat. ⏱️ Cooldown Aware - Reports cooldown time instead of trying unusable cloaks. 🌍 Localized - Supports major WoW locales automatically. 🧭 Movable Minimap Icon - Uses the CCU logo and can be right-dragged around the minimap while left-click keeps your teleport flow close at hand. **Supported Cloaks:** Cloak of Coordination, Wrap of Unity, Shroud of Cooperation and their variants. *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/ccu # Trigger cloak detection and show the use button
/ccu help # Show command help
/ccu welcome # Toggle login welcome message
